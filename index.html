Lenguajes de Programación Favoritos

Una breve descripción de mis tres lenguajes preferidos y para qué sirven, pensada para entornos de accesibilidad y diseño web.

  1. JavaScript

    ¿Para qué sirve? Es el alma interactiva de la web. Permite crear funciones dinámicas como menús desplegables, sliders, formularios inteligentes, animaciones y más.

    Ideal para usabilidad y prototipos interactivos.

  2. HTML (HyperText Markup Language)

    ¿Para qué sirve? Define la estructura básica de una página web: títulos, párrafos, botones, formularios, imágenes y más.

    Clave para aplicar principios de accesibilidad como la semántica y el orden lógico.

  3. Python

    ¿Para qué sirve? Lenguaje versátil y legible, usado para automatización, análisis de datos, desarrollo back-end y scripts de optimización.

    Útil para análisis de comportamiento del usuario y pruebas de accesibilidad.
